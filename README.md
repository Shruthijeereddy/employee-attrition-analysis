# Employee Attrition Analysis

## Overview
This project performs Exploratory Data Analysis (EDA) on an HR employee dataset to understand patterns influencing employee attrition. The analysis uses Python and data visualization techniques to identify trends related to departments, job roles, income, and job satisfaction.

## Business Problem
Employee attrition is a major challenge for organizations as it affects productivity and hiring costs. The goal of this project is to analyze workforce data and identify factors associated with employee turnover.

## Solution Approach
- Loaded and explored HR dataset using Pandas
- Performed data cleaning and preprocessing
- Created visualizations to analyze attrition trends
- Generated business insights using statistical summaries

## Tools & Technologies
- Python
- Pandas
- Matplotlib
- Seaborn
- Data Visualization

## Key Analysis Performed
- Attrition distribution analysis
- Department-wise attrition comparison
- Salary distribution visualization
- Job satisfaction impact analysis
- Job role attrition patterns
- Feature correlation heatmap

## Results & Insights
- Identified departments and job roles with higher attrition
- Observed relationships between satisfaction levels and employee turnover
- Visualized income distribution patterns across employees
